2. How We Use Your Information

  • To execute and respond to user commands
  • To enable and support persistent features, including:

    • Call history and connection metadata (!gwagwa call)

We do not use your data for advertising or profiling. Any stored data is only used internally to support and maintain the features you choose to use.

3. Data Storage

GwaGwa stores data only when necessary to fulfill the functionality of specific commands and features. Persistent data includes:

  • Channel configurations: Server IDs and channel IDs mapped for cross-server communication (!gwagwa channelset).
  • Call logs: Metadata such as server IDs, timestamps, and call statuses for connections initiated via !gwagwa call.
  • Job application records: Job names, user IDs, and status flags when you apply (!gwagwa apply) or quit (!gwagwa quit).
  • Rizz points ledger: Cumulative rizz point totals per user, adjusted via !gwagwa rizzadd or !gwagwa rizzremove.

All stored data is retained securely following best practices in encryption and access control. We will notify you in advance if any additional features require the collection or storage of new types of personal data.
