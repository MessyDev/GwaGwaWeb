GwaGwa Bot Guide & Commands

Cross-Server Communication Discord Bot by abcman123

Welcome to the official guide for GwaGwa Bot. This document outlines setup instructions, available commands, and usage examples. Follow each section carefully to configure the bot correctly and leverage its full capabilities.

1. Bot Setup

  1. Invite the Bot: Use the OAuth2 link with the following minimum permissions:
    • Send Messages
    • Manage Channels (for cross-server linking)
    • Read Message History
    • Embed Links
  2. Verify Permissions: Ensure that the bot’s role is positioned above any roles it should manage.
  3. Configure Channel: (Detailed in !gwagwa channelset) Set the primary relay channel once the bot is in your server.

2. Command Prefix

All commands for GwaGwa Bot begin with the prefix: !gwagwa

3. Facility Commands

  • !gwagwa facility jobs
    Display a list of all available jobs and their respective hourly pay rates.
  • !gwagwa facility stats
    Show dynamic facility statistics (e.g., cleanliness, reactor level, security status, employment rate).
  • !gwagwa facility perks
    List server perks unlocked based on chat activity and number of filled job slots.

4. Job Application Commands

  • !gwagwa apply <job_name>
    Submit an application for a specific job (only available if slots are open).
  • !gwagwa quit
    Resign from your current job position.

5. Rizz Points Commands

  • !gwagwa rizzpoints [<@user>]
    Check your own rizz points, or view another user’s total if a mention is provided.
  • !gwagwa rizzadd <@user> <amount>
    (Admin Only) Award a specified number of rizz points to a user.
  • !gwagwa rizzremove <@user> <amount>
    (Admin Only) Deduct a specified number of rizz points from a user.

6. Cross-Server Communication

  • !gwagwa channelset <channel_id|channel_name|<#channel_id>>
    (Admin Only) Designate a channel in this server for incoming relayed messages. Once set, any connected server using !gwagwa call will broadcast into this channel.
  • !gwagwa call <server_id>
    Initiate a connection request to a target server. The receiving server will see interactive buttons to accept or decline the request.
  • !gwagwa call hangup
    Terminate an active cross-server connection (“hang up”).

7. Utility & Server Info Commands

  • !gwagwa servers
    List all servers that GwaGwa Bot is currently in, along with their server IDs.

8. Bot Owner & Admin-Only Commands

  • !gwagwa notify <message>
    (Bot Owner Only) Broadcast a custom message to every channel set via !gwagwa channelset.
  • !gwagwa rizzadd <@user> <amount>
    (Admin Only) (Same as above in Rizz Points section.)
  • !gwagwa rizzremove <@user> <amount>
    (Admin Only) (Same as above in Rizz Points section.)
  • !gwagwa channelset <channel_id|channel_name|<#channel_id>>
    (Admin Only) (As described in Cross-Server Communication.)

9. Informational & Policy Commands

  • !gwagwa termsofservice
    Show the Terms of Service link for GwaGwa Bot.
  • !gwagwa privacypolicy
    Show the Privacy Policy link for GwaGwa Bot.
  • !gwagwa guide
    Display the link to this Guide/documentation page.
  • !gwagwa help
    Show the complete help menu (all commands with brief descriptions).

10. Examples


// 1. Apply for the "Mechanic" job:
!gwagwa apply mechanic

// 2. Check Facility Stats:
!gwagwa facility stats

// 3. Admin sets this channel for cross-server relay:
!gwagwa channelset 123456789012345678

// 4. Start a cross-server call to server ID 987654321098765432:
!gwagwa call 987654321098765432

// 5. Hang up an active call:
!gwagwa call hangup

// 6. Check your own Rizz points:
!gwagwa rizzpoints

// 7. Admin adds 50 Rizz points to @User:
!gwagwa rizzadd @User 50

// 8. List all servers GwaGwa is in:
!gwagwa servers

// 9. Bot owner sends a notification to all linked channels:
!gwagwa notify Attention all servers, maintenance starts in 10 minutes!

// 10. View Terms of Service:
!gwagwa termsofservice

// 11. Show this guide:
!gwagwa guide

// 12. Display help menu:
!gwagwa help
