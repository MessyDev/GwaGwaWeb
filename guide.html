This guide covers setup, commands, and examples for GwaGwa, the cross-server communication Discord bot by abcman123.

1. Bot Setup

    Join the bot to your server.

2. Command Prefix

All commands use the prefix: !gwagwa

3. Facility Commands

  • !gwagwa facility jobs
    List all available jobs and hourly pay rates.
  • !gwagwa facility stats
    Show dynamic facility stats (cleanliness, reactor level, security, employment rate).
  • !gwagwa facility perks
    Display server perks based on chat activity and jobs filled.

4. Job Application Commands

  • !gwagwa apply <job name>
    Apply for a job (if slots available).
  • !gwagwa quit
    Quit your current job.

5. Rizz Points Commands

  • !gwagwa rizzpoints [<@user>]
    Check your or another user's rizz points.
  • !gwagwa rizzadd <@user> <amount>
    *(Admin only)* Add rizz points to a user.
  • !gwagwa rizzremove <@user> <amount>
    *(Admin only)* Remove rizz points.

6. Cross-Server Communication

  • !gwagwa channelset <channel_id>
    *(Admin only)* Set the channel for relayed messages in this server.
  • !gwagwa call <server_id>
    Initiate a call to another server with buttons to accept or decline.
  • !gwagwa call hangup
    End an active call.

7. Utility Commands

  • !gwagwa servers
    List all servers the bot is currently in.

8. Examples

// Checking facility stats
!gwagwa facility stats
