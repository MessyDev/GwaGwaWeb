This guide covers setup, commands, and examples for GwaGwa, the cross-server communication Discord bot by abcman123.

1. Bot Setup

  1. Invite the bot to your server (use the OAuth2 link with proper permissions).
  2. Ensure the bot has these minimum permissions:
    • Send Messages
    • Manage Channels (for cross-server linking)
    • Read Message History
    • Embed Links

2. Command Prefix

All commands use the prefix: !gwagwa

3. Facility Commands

  • !gwagwa facility jobs
    List all available jobs and their hourly pay rates.
  • !gwagwa facility stats
    Show dynamic facility stats (cleanliness, reactor level, security status, employment rate).
  • !gwagwa facility perks
    Display server perks based on chat activity and number of jobs filled.

4. Job Application Commands

  • !gwagwa apply <job_name>
    Apply for a job (if there are open slots for that position).
  • !gwagwa quit
    Quit your current job.

5. Rizz Points Commands

  • !gwagwa rizzpoints [<@user>]
    Check your own or another user's rizz points.
  • !gwagwa rizzadd <@user> <amount>
    (Admin only) Add rizz points to a given user.
  • !gwagwa rizzremove <@user> <amount>
    (Admin only) Remove rizz points from a given user.

6. Cross-Server Communication

  • !gwagwa channelset <channel_id|channel_name|<#channel_id>>
    (Admin only) Set the designated channel in this server for relayed messages. Once set, any server connected via !gwagwa call will broadcast into this channel.
  • !gwagwa call <server_id>
    Initiate a cross-server connection request to the target server. The other server will see buttons to accept or decline.
  • !gwagwa call hangup
    End an active cross-server connection (hang up).

7. Utility & Server Info Commands

  • !gwagwa servers
    List all the servers that GwaGwa Bot is currently in (along with their IDs).

8. Bot Owner & Admin-Only Commands

  • !gwagwa notify <message>
    (Bot Owner only) Send a broadcast message to all channels that have been set via !gwagwa channelset.
  • !gwagwa rizzadd <@user> <amount>
    (Admin only) Add rizz points to a user (listed above in Rizz Points).
  • !gwagwa rizzremove <@user> <amount>
    (Admin only) Remove rizz points from a user (listed above in Rizz Points).
  • !gwagwa channelset <channel_id|channel_name|<#channel_id>>
    (Admin only) (also listed above under Cross-Server)

9. Informational & Policy Commands

  • !gwagwa termsofservice
    Display the Terms of Service link for GwaGwa Bot.
  • !gwagwa privacypolicy
    Display the Privacy Policy link for GwaGwa Bot.
  • !gwagwa guide
    Display the link to this Guide page/documentation.
  • !gwagwa help
    Show the complete help message (all commands and a brief description).

10. Examples

// 1. Applying for the "Mechanic" job
!gwagwa apply mechanic

// 2. Checking facility stats
!gwagwa facility stats

// 3. Admin sets this channel for cross-server relay
!gwagwa channelset 123456789012345678

// 4. Starting a cross-server call to server ID 987654321098765432
!gwagwa call 987654321098765432

// 5. Hanging up an active call
!gwagwa call hangup

// 6. Checking your own rizz points
!gwagwa rizzpoints

// 7. Admin adding 50 rizz points to @User
!gwagwa rizzadd @User 50

// 8. Listing all servers GwaGwa is in
!gwagwa servers

// 9. Bot owner sending a notification to all linked channels
!gwagwa notify Attention all servers, maintenance starts in 10 minutes!

// 10. Viewing Terms of Service
!gwagwa termsofservice

// 11. Showing this guide
!gwagwa guide

// 12. Getting the help menu
!gwagwa help
